### [NLW Heat] Mission: Impulse - React

[Vite](https://vitejs.dev/guide/)

##### Criando o Projeto
```
yarn create vite react --template react-ts
```

##### Instalação do SASS
```
yarn add sass
```

##### Instalação do React Icons
```
yarn add react-icons
```

##### Instalação do axios
```
yarn add axios
```

##### Instalação do socket.io-client
```
yarn add socket.io-client
```
